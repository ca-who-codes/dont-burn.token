# Skill Index

Read this before loading any skill file. Match the task description to an entry, then load only the matching skill file.

| Skill | File | Load when the task involves... |
|-------|------|-------------------------------|
| code | skills/code.md | reading, writing, debugging, refactoring, or reviewing source code |
| web | skills/web.md | browser automation, HTTP requests, scraping, or API testing |
| data | skills/data.md | data analysis, SQL queries, pandas/polars operations, or charts |
| write | skills/write.md | documentation, commit messages, emails, specs, or PRDs |

## Rules

- A task can match multiple skills — load all matches
- If no skill matches, proceed without loading any skill file
- Load skills sequentially, not all at once
- After loading a skill, do not re-read this index
